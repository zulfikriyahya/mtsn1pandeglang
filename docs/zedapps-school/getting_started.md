# Getting Started

<div style = {{ textAlign: 'justify' }}>
Markdown adalah alat yang sangat fleksibel untuk menulis dokumentasi yang menyertakan kode. Berikut adalah beberapa bahasa pemrograman yang didukung untuk penyorotan sintaks dalam Markdown:
</div>

## Markdown Code

### Bash

    ```bash title="Bash"
    echo "Hello, world!"
    ```

### Python

    ```python title="Python"
    def hello_world():
        print("Hello, world!")
    ```

### JavaScript

    ```javascript title="Javascript"
    function helloWorld() {
      console.log("Hello, world!");
    }
    ```

### Java

    ```java title="Java"
    public class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, world!");
        }
    }
    ```

### C++

    ```cpp title="C++"
    #include <iostream>

    int main() {
        std::cout << "Hello, world!" << std::endl;
        return 0;
    }
    ```

### HTML

    ```html title="HTML"
    <!DOCTYPE html>
    <html>
      <head>
        <title>Hello World</title>
      </head>
      <body>
        <h1>Hello, world!</h1>
      </body>
    </html>
    ```

### CSS

    ```css title="CSS"
    h1 {
      color: blue;
    }
    ```

### SQL

    ```sql title="SQL"
    SELECT * FROM users WHERE active = 1;
    ```

### Ruby

    ```ruby title="Ruby"
    puts 'Hello, world!'
    ```

### PHP

        ```php title="PHP"
        <?php
        echo "Hello, world!";
        ?>
        ```

    :::info

<div style = {{ textAlign: 'justify' }}>
    Ini hanya beberapa contoh bahasa pemrograman yang didukung. Anda dapat menggunakan tanda backtick tiga kali diikuti dengan nama bahasa pemrograman untuk mendapatkan penyorotan sintaks yang sesuai di Markdown. 🚀
</div>
    :::
