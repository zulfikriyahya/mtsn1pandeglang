# MTs Negeri 1 Pandeglang

Portal resmi **Madrasah Tsanawiyah Negeri 1 Pandeglang** yang dibangun dengan teknologi modern untuk memberikan informasi dan layanan terbaik kepada civitas akademika dan masyarakat.

## Deskripsi Proyek

Website ini merupakan platform digital komprehensif yang menyediakan informasi lengkap tentang madrasah, kegiatan akademik, prestasi siswa, serta berbagai layanan terpadu untuk mendukung ekosistem pendidikan yang transparan dan efisien.

**Moto:** Mandiri, Takwa, Peduli Lingkungan, Prestasi

## Teknologi Utama

- **Astro 5.15.4** - Static Site Generator untuk performa optimal
- **React 19.2.0** - Library UI untuk komponen interaktif
- **TailwindCSS 4.1.14** - Framework CSS utility-first
- **TypeScript 5.9.3** - Type-safe JavaScript
- **MDX** - Markdown dengan kemampuan JSX
- **PWA Support** - Progressive Web App capability

## Instalasi

### Prasyarat

- Node.js versi 18.x atau lebih tinggi
- Yarn package manager

### Langkah Instalasi

```bash
# Clone repository
git clone https://github.com/zulfikriyahya/mtsn1pandeglang.git

# Masuk ke direktori proyek
cd mtsn1pandeglang

# Install dependencies
yarn install

# Generate JSON files
yarn generate-json

# Jalankan development server
yarn dev
```

Website akan berjalan di `http://localhost:4321`

## Perintah Tersedia

```bash
# Development
yarn dev                 # Jalankan development server
yarn build              # Build untuk production
yarn preview            # Preview production build
yarn check              # Type checking dengan Astro

# Maintenance
yarn format             # Format code dengan Prettier
yarn generate-json      # Generate search index
yarn remove-darkmode    # Remove dark mode (opsional)
```

## Konfigurasi

### Site Configuration (`src/config/config.json`)

```json
{
  "site": {
    "title": "MTs Negeri 1 Pandeglang",
    "base_url": "https://mtsn1pandeglang.sch.id",
    "favicon": "/images/favicon.png",
    "logo": "/images/brand-lightmode.png"
  },
  "settings": {
    "search": true,
    "sticky_header": true,
    "theme_switcher": true,
    "pagination": 10
  }
}
```

### Theme Configuration (`src/config/theme.json`)

Sesuaikan warna, font, dan styling sesuai kebutuhan madrasah.

## Content Management

### Menambah Post Blog

Buat file baru di `src/content/blog/` dengan format:

```markdown
---
title: "Judul Artikel"
description: "Deskripsi singkat"
date: 2025-01-10T08:00:00Z
image: "/images/artikel/gambar.png"
categories: ["Kategori"]
author: "Nama Penulis"
tags: ["tag1", "tag2"]
draft: false
---

Konten artikel...
```

### Menambah Halaman Statis

Buat file di `src/content/pages/` dengan struktur serupa.

### Mengelola Menu

Edit `src/config/menu.json` untuk mengatur navigasi.
